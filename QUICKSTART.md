# Быстрый старт

## Запуск сервера

```bash
cd Server
dotnet run
```

Сервер запустится на `localhost:5000` и будет ждать подключения игроков.

## Запуск клиента (Windows/Visual Studio)

1. Откройте решение `OrisSemesterWork2.slnx` в Visual Studio
2. Установите проект `Client` как стартовый
3. Запустите (F5)

## Запуск клиента (macOS/Linux - не поддерживается)

WinForms работает только на Windows. Для тестирования на macOS/Linux можно:
- Использовать Windows VM
- Или создать консольный клиент для тестирования

## Как играть

1. Запустите сервер
2. Запустите 2-4 клиента
3. В каждом клиенте введите:
   - Никнейм
   - Email
   - Адрес сервера (по умолчанию localhost:5000)
4. Нажмите "Подключиться"
5. Когда подключится минимум 2 игрока, игра начнется автоматически
6. Выберите архетип или откажитесь (играть нейтрально)
7. Когда придет ваш ход:
   - Смотрите результаты производства в логе
   - Стройте здания (выберите клетку на поле, выберите тип здания, нажмите "Построить")
   - Улучшайте здания (выберите клетку с зданием, нажмите "Улучшить")
   - Производите солдат (введите количество, нажмите "Произвести")
   - Атакуйте противников (выберите цель, количество солдат, нажмите "Атаковать")
   - Завершите ход кнопкой "ЗАВЕРШИТЬ ХОД"

## Основные правила

- Игра длится 15 циклов
- В каждом цикле все игроки делают по одному ходу в случайном порядке
- Атаки запрещены первые 5 циклов
- Побеждает игрок с наибольшим количеством очков за ресурсы
- Солдаты не дают очков

## Типы зданий

### Производители (зеленые)
- Лесозаготовка - производит Дерево
- Карьер - производит Камень
- Шахта - производит Руду
- Поле - производит Зерно

### Переработчики класс 2 (голубые)
- Лесопилка - Дерево → Доски
- Печь обжига - Камень → Кирпичи
- Плавильня - Руда → Металл
- Углежог - Дерево → Уголь
- Дробилка - Камень → Песок
- Пекарня - Зерно + Дерево → Хлеб

### Переработчики класс 3 (коралловые)
- Столярная - Доски → Мебель
- Каменотёс - Кирпичи + Камень → Стены
- Кузница - Металл + Дерево → Инструменты
- Стекловарня - Песок + Уголь → Стекло
- Оружейная - Металл + Доски → Оружие

### Драгоценности (золотые)
- Лаборатория - Стекло + Уголь + Инструменты → Изумруд
- Алхимическая печь - Металл + Уголь + Инструменты → Золото

### Военные (оранжевые)
- Казармы - Хлеб + Оружие → Солдаты

### Оборона (коричневые)
- Баррикада - +5% обороны (улучшение: +10%)
- Оборонительная башня - +20% обороны (улучшение: +30%)

## Архетипы

- **Жадина**: +30% оборона, -20% очков
- **Меценат**: +25% очков, -25% оборона
- **Воин**: Игнорирует 20% обороны, дорогие солдаты (5 хлеб + 2 оружие)
- **Новобранец**: Дешевые солдаты (1 хлеб + 1 оружие), страдают от обороны
- **Инженер**: Возврат 1-2 ресурсов при строительстве, -20% очков
- **Алхимик**: +25% очков за драгоценности, -30% за остальное
- **Обжора**: Солдаты крадут 2 ресурса, едят 6 хлеб + 1 оружие
- **Нейтральный**: Без модификаторов

## Советы

1. Начните с производителей базовых ресурсов
2. Постройте переработчики для получения продвинутых ресурсов
3. Не забывайте про оборону - она защищает от атак
4. Драгоценности дают много очков, но требуют сложной цепочки производства
5. Солдаты не дают очков, но позволяют красть ресурсы у противников
6. Улучшайте здания для увеличения производства
