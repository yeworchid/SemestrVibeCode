Каждый игрок совершает ход в свою очередь. Ход состоит из **пяти фаз**, из которых **фазы 3–5 — опциональные** (можно пропустить).

### Фаза 1. **Производство**  
Все **производители** на вашем поле генерируют ресурсы.  
Обработка идёт **слева направо, сверху вниз**.

### Фаза 2. **Переработка**  
Все **переработчики** выполняют преобразования, если хватает входных ресурсов.  
Обработка — в том же порядке (слева→направо, сверху→вниз).  
Каждое здание может переработать **не больше своего лимита** (зависит от уровня).

### Фаза 3. **Производство Солдат** *(опционально)*  
Вы можете произвести **Солдат** в Казармах, потратив указанные ресурсы.  
Количество за ход ограничено уровнем Казарм.

### Фаза 4. **Атака** *(опционально)*  
- Можно атаковать **только одного** игрока.  
- **Первые 5 циклов атака запрещена** (все под защитой).  
- Вы отправляете любое количество Солдат.  
- Цель теряет **случайные ресурсы** — по одному за каждого **дожившего** Солдата.  
- Если у цели **нет ресурсов**, Солдат возвращается без добычи.  
- **Потери** = ⌈отправлено × (% обороны цели / 100)⌉ → округляется **вверх**.  
- Вернувшиеся Солдаты можно использовать снова.

### Фаза 5. **Строительство** *(опционально)*  
Вы можете:  
- Построить новое здание (за ресурсы).  
- Улучшить существующее (если прошёл хотя бы один ход после постройки).
